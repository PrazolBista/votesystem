# Readme file for Voting Management System

This is a guide on how to install and set up the Voting Management System on XAMPP. Follow the steps below to get started.

## Prerequisites

1. XAMPP
2. Text editor such as Notepad++ or Sublime Text 3 / etc.

## Installation

1. Download the zip file or download WinRAR
2. Extract the file and copy "votesystem" folder
3. Paste inside root directory\htdocs where you install XAMPP: local disk C: drive D: drive E: paste: (for XAMPP/htdocs)
4. Open PHPMyAdmin (http://localhost/phpmyadmin)
5. Create a database with name votesystem
6. Import votesystem.sql file (given inside the zip package in SQL file folder)
7. Run the script http://localhost/voting management system
